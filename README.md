# Actividad 2: Creación de DataFrames con Pandas

Este proyecto de Streamlit es una actividad práctica diseñada para aprender y aplicar diferentes métodos de creación y carga de DataFrames utilizando la librería Pandas en Python.

## 🚀 Cómo ejecutar el proyecto

Para ejecutar esta aplicación en tu entorno local, sigue estos pasos:

1. **Clona el repositorio** o descarga los archivos.
2. Abre una terminal en la carpeta principal del proyecto.
3. **Instala las dependencias** necesarias (se recomienda usar un entorno virtual):
   ```bash
   pip install -r requirements.txt
   ```
   *(Nota: Asegúrate de tener instalados `streamlit`, `pandas`, `requests` y `openpyxl` si no usas el requirements)*
4. **Ejecuta la aplicación Streamlit** con el siguiente comando:
   ```bash
   streamlit run Inicio.py
   ```
5. Abre el enlace local proporcionado (generalmente `http://localhost:8501`) en tu navegador web.

## 👩‍💻 ¿Qué debes hacer como estudiante?

Tu misión es resolver los ejercicios de código ubicados en cada una de las páginas de la aplicación. 

En la interfaz de Streamlit (menú lateral a la izquierda) verás diferentes secciones. **Por cada sección, debes abrir el archivo de código correspondiente e ingresar tu solución.** Busca el comentario `# ESTUDIANTE: Escribe tu código a continuación` en cada archivo.

Aquí tienes la ruta exacta de los archivos que debes modificar:

1. **Series ➡️** Crea un DataFrame uniendo varias Series.
   📍 Edita el archivo: `pages/1_Series.py`

2. **Listas ➡️** Forma un DataFrame usando una lista de listas.
   📍 Edita el archivo: `pages/2_Listas.py`

3. **CSV ➡️** Carga y visualiza un CSV local y uno desde una URL de internet.
   📍 Edita el archivo: `pages/3_CSV.py`

4. **Excel ➡️** Lee un archivo con extensión `.xlsx`.
   📍 Edita el archivo: `pages/4_Excel.py`

5. **JSON ➡️** Carga información desde un archivo `.json` local.
   📍 Edita el archivo: `pages/5_JSON.py`

6. **API ➡️** Haz una petición `GET` a la PokéAPI y convierte los resultados de la API en una tabla.
   📍 Edita el archivo: `pages/6_API.py`

7. **MongoDB ➡️** Muestra el código teórico para extraer datos de una colección en MongoDB Atlas.
   📍 Edita el archivo: `pages/7_MongoDB.py`

8. **Firebase ➡️** Proporciona el código de conexión y extracción de base de datos desde Firestore hacia Pandas.
   📍 Edita el archivo: `pages/8_Firebase.py`

¡Lee atentamente las instrucciones en la pantalla de cada ejercicio y diviértete programando!
